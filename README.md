# rite

A basic let-go application.

## Development

Install dependencies with [mise](https://mise.jdx.dev/getting-started.html) (or manaully consulting the `.mise.toml` file):

```bash
mise trust && mise install
```

Run main application commands:
```bash
lgx --help
lgx run
lgx test
lgx fmt
lgx lint
lgx nrepl
lgx build
```
